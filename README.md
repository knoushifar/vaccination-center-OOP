
# VaccinationCenter

## Overview
This is a Java-based program that simulates the management of a vaccination center.
It demonstrates object-oriented programming concepts such as classes, objects, and encapsulation.

## Features
- Manage vaccination appointments.
- Track vaccinated individuals.
- Simple console-based interface for demonstration.

## Structure
- src/VaccinationCenter.java: Main source code of the project.

## Requirements
- Java 8 or above
- Command-line interface to run the program

## How to Run
1. Clone the repository:
   git clone https://github.com/knoushifar/vaccination-center-OOP.git
2. Navigate to the source folder:
   cd vaccination-center-OOP/src
3. Compile and run:
   javac VaccinationCenter.java
   java Main
